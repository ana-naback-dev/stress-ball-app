# Stress Ball App 🎯

## 🚀 Acesse o projeto

👉 https://ana-naback-dev.github.io/stress-ball-app/
Interactive application designed to help users relieve stress through a virtual stress ball.

## 🚀 Features

- Interactive stress ball
- Animation when clicking the ball
- Simple and intuitive interface

## 🛠️ Technologies

- HTML
- CSS
- JavaScript

## ▶️ How to run

1. Clone this repository:
   git clone https://github.com/ana-naback-dev/stress-ball-app.git

2. Open the `index.html` file in your browser

This project was created to practice web development concepts and user interaction.
